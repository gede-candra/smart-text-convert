Smart Text Convert

Toolkit cepat untuk command shell, formatting list, pembersihan, perbandingan, dan hashing teks.

Menu Fitur

Find Command: Kecualikan Daftar

Masukkan nama file per baris. Tool ini akan generate command `find` untuk menampilkan (`-print`), menghitung jumlah (`| wc -l`), dan menghapus (`-delete`) file selain nama yang Anda tulis.

Command preview
find . -maxdepth 1 -type f -print
Command delete
find . -maxdepth 1 -type f -delete
Command count
find . -maxdepth 1 -type f -print | wc -l

Opsi jika line sangat banyak (pakai file txt):

Contoh dengan `keep.txt` (1 nama file per baris)
# lihat file yang akan terpilih find . -maxdepth 1 -type f -printf '%f\n' | grep -Fvx -f keep.txt # hitung jumlah file find . -maxdepth 1 -type f -printf '%f\n' | grep -Fvx -f keep.txt | wc -l # hapus file selain yang ada di keep.txt find . -maxdepth 1 -type f -printf '%f\0' | grep -zFvx -f keep.txt | xargs -0 -r -I{} rm -- "./{}"
Catatan: nama file otomatis di-escape jika mengandung tanda petik satu.

Find Command: Pilih Hanya Daftar

Masukkan nama file per baris. Tool ini akan generate command `find` untuk memilih hanya file yang Anda tulis (kebalikan dari fitur Find Command utama).

Command preview
find . -maxdepth 1 -type f -false -print
Command delete
find . -maxdepth 1 -type f -false -delete
Command count
find . -maxdepth 1 -type f -false -print | wc -l

Opsi jika line sangat banyak (pakai file txt):

Contoh dengan `pick.txt` (1 nama file per baris)
# lihat file yang masuk daftar pick find . -maxdepth 1 -type f -printf '%f\n' | grep -Fx -f pick.txt # hitung jumlah file yang masuk daftar pick find . -maxdepth 1 -type f -printf '%f\n' | grep -Fx -f pick.txt | wc -l # hapus hanya file yang ada di pick.txt find . -maxdepth 1 -type f -printf '%f\0' | grep -zFx -f pick.txt | xargs -0 -r -I{} rm -- "./{}"
Catatan: jika input kosong, command dibuat aman dengan `-false` (tidak memilih file).
