# Smart Text Convert

Smart Text Convert is a lightweight web utility that helps you convert text into terminal-ready commands and practical text formats.

This project is a dependency-free single-page app built with plain HTML, CSS, and JavaScript.

For Indonesian documentation, see [README.id.md](README.id.md).

## Features

### 1. Find: Exclude List
Generates `find` commands to target files **not included** in your input list.

Available outputs:
- `-print` (preview files)
- `-delete` (delete files)
- `| wc -l` (count files)

Extra:
- File-based option using `keep.txt` for long lists.

### 2. Find: Include Only List
The inverse mode: generates `find` commands to target **only** files included in your input list.

Available outputs:
- `-print`
- `-delete`
- `| wc -l`

Extra:
- File-based option using `pick.txt` for long lists.

### 3. Text List Formatter
Converts multiline input into different list formats.

Formatting options:
- separator: `space`
- separator: `,`
- wrapper: `""`
- wrapper: `''`

Example outputs:
- `a, b, c, d`
- `"a", "b", "c", "d"`

### 4. Duplicate Line Cleaner
Removes duplicate lines while preserving the first occurrence order.

Extra:
- shows input line count
- shows output line count

### 5. Compare Missing Lines
Compares two text blocks and shows which lines are missing on each side.

Available outputs:
- lines present in Text A but missing in Text B
- lines present in Text B but missing in Text A

Extra:
- shows unique line count for each side
- shows missing line count for each side
- copy button for each comparison result

Comparison behavior:
- compares by unique lines only (duplicates are ignored)
- trims leading/trailing spaces per line
- ignores empty lines

## UX Highlights

- Sidebar navigation for quick feature switching.
- Scrollable output blocks for large results.
- Copy buttons for all major outputs.

## Run Locally

### Option 1: Open directly in browser
Open `index.html` directly.

### Option 2: Run with Live Server (recommended)
1. Open this project in VS Code.
2. Install the `Live Server` extension (by Ritwick Dey) if you do not have it yet.
3. Right-click `index.html`.
4. Click `Open with Live Server`.

Then open the local URL generated by Live Server (usually `http://127.0.0.1:5500`).

## Safety Notes

If you use commands containing `-delete`:
1. Run the `-print` version first.
2. Check total files with `| wc -l` before deletion.
3. For very long lists, use file-based mode (`keep.txt` / `pick.txt`) to avoid command-length limits.
